For Background Tasks (Upstash QStash):

1) Sign up at https://upstash.com/ and create a QStash project
2) Get your QSTASH_TOKEN from the dashboard
3) Set environment variable: QSTASH_TOKEN=your_token_here
4) Configure scheduled tasks (one-time): python setup_qstash_schedules.py
5) Tasks are automatically called via HTTP endpoints by QStash

Task Endpoints:
- /api/tasks/send-email/ - Send email asynchronously
- /api/tasks/generate-qr-code/ - Generate QR code
- /api/tasks/monthly-bill-check/ - Daily at 23:00 (scheduled)
- /api/tasks/reset-dashboard/ - Daily at 00:01 (scheduled)
- /api/tasks/monthly-batch-processing/ - Monthly on day 1 at 03:00 (scheduled)

For more info: https://upstash.com/docs/qstash/overall/getstarted
